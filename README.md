# VCS Assignment Project

## Overview
This project was created for my Version Control Systems assignment. It demonstrates Git basics, branching, merge conflicts, pull requests, and documentation practices.

## How to Run
1. Make sure Python is installed
2. Clone the repo:
git clone https://github.com/mvgrn/vcs-assignment.git
3. Navigate to folder:
cd vcs-assignment
4. Run the program:
python src/main.py
